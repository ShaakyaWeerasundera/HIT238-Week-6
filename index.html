Semantic HTML

The correct way to use HTML. HTML that is Semantic helps search engines and screen headers

Linking CSS

Use the "link rel="stylesheet"" tag at the top of the HTML document after the title tag.

Linking Javascript

Use the "script src="javascript"" tag at the bottom of the HTML document (after the body tag)

meta tag for mobile screens

The viewport tag. eg. meta name="viewport" content="width=device-width", initial-scale=1.0

CSS for certain screen sizes

@media queries can be used to apply css to certain screen sizes

Mobile First

Focuses more on the the mobile device CSS than the Desktop CSS.

Applying CSS Order

The selectors (Classes and ID) have certain precedences. The highest precedence selector has the the style applied to it.

JS Best Practices

Avoid global variables, use === to compare values, avoid the use of eval().

JS Buttons

Use document.addEventListener() functions with click and the function being refered to as arguments

== vs ===

== ignores the datatype of the variable. === checks the datatype and compares 2 values.
